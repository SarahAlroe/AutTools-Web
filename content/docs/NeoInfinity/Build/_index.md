# How to make a badge

Making a NeoInfinity badge is relatively simple: It needs to be soldered and programmed, and then a fitting power supply must be sourced. For anyone familiar with Arduino and SMD soldering, this should be a walk in the park. The following pages are a comprehensive guide on how to accomplish the above. Follow, deviate and skip as necessary. 

Time estimate of active work: 1 hour. (Can be reduced to about 20 minutes with some practice if doing batch production.) 
Time estimate including waiting for parts: A couple of weeks. 
Breakdown:
- PCB production and component ordering (waiting): 1-3 weeks.
- PCB assembly: 15-45 minutes.
- Programming and configuration: 10 minutes.

All required PCB files and code for producing a badge is available in the [NeoInfinity](https://github.com/SarahAlroe/NeoInfinity) repository.
