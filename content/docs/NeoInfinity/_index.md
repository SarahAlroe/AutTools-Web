# NeoInfinity: An electronic badge that lights up in the colors of the spectrum

The NeoInfinity badge is a physical recreation of the neurodiversity infinity symbol, using RGB LEDs to animate an infinitely cycling rainbow. The badge can also be used as an [interaction signaling badge](https://stimpunks.org/access/interaction/) by showing red, yellow and green color patterns. The badge is controlled by a button on the back, which can be pressed to cycle through colors, animation patterns and speeds, and held down to change the brightness. The badge runs on anything from 2.5-5.5V, so can be driven off USB power, a LiPo cell or 3-4 AA batteries (whether alkaline or NiMH).
