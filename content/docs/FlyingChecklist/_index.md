# A Flying Checklist

The flying checklist is a digital hybrid between a dictaphone and a written checklist, a cognitive support tool for helping remember (and forget) things: 
By activating its microphone, the device can record short phrases which are then transcribed and added to the checklist. 
Items on the list can then be checked off as complete, which is then automatically re-organized to maximize overview.
The flying checklist is currently under active development, is currently being trialed.
