# Contributing to the project
If you make any modifications, additions or changes to your AutCorder or NeoInifinity badge which others may find useful, be it hardware revisions, new shells, additional software functionality, guides for modification or suggestions for use, please open a pull request in the relevant repository or [send an email to Sarah](mailto:salroee@cc.au.dk?subject=AutTools). Partial, or rough additions are very welcome, as are plain ideas, requests or suggestions!
